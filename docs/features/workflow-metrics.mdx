# Workflow metrics

![Workflow metrics](/img/features/wf-metrics-cut.gif)

Workflow metrics will help you to take control of your workflow performance. You can sisualize metrics such as CPU Load, Memory usage, Network and Disk I/O
for each workflow run. To enable Process Trace Chart you need to integrate [Workflow Kit](/advanced-configuration/workflow-kit).
