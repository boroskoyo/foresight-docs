Process trace chart will help you to understand what happens on your workflow runs at Kernel level. 
To enable Process Trace Chart you need to integrate [Workflow Kit](/advanced-configuration/workflow-kit).

:::info
Process trace chart works only on the Ubuntu
:::

### Critical path
On the process trace map, you see that there is white lines inside the span. It is called "Critical Path".
It indicates the time spend by the process itself not their child processes.
